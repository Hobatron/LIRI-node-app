# LIRI-node-app
A Spotify, Concert, and OMDB search engine

## Install:
  1. git clone git@github.com:Hobatron/LIRI-node-app.git
  2. cd LIRI-node-app
  3. npm i
  
## Run:
  1. npm server.js
